HTML

Que es el HTML

HTML ( HyperText Markup Language) es el lenguaje estándar de marcado utilizado para
crear y estructurar el contenido de páginas web.
Mediante etiquetas (<>), organiza textos, imágenes, videos y enlaces
que los navegadores interpretan para mostrar el sitio web.
Es la estructura básica y esencial de internet, no un lenguaje de programación.

El html es el lenguaje de marcado de hipertexto

Historia de HTML

HTML fue creado en 1991 por Tim Berners-Lee para crear páginas web. Con el tiempo evolucionó hasta HTML5, permitiendo agregar imágenes, videos, formularios y contenido interactivo en las páginas web.

html5

HTML5 es la última versión de HTML, lanzada en 2014.
Introdujo nuevas etiquetas como para mejorar la estructura de las páginas web.
También agregó soporte para multimedia, gráficos y aplicaciones web,
convirtiéndose en el estándar actual para el desarrollo web.
