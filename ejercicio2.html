Que es el HTML

HTML (Lenguaje de Marcado de Hipertexto) es el lenguaje estándar para crear y estructurar páginas web. Define la estructura básica del contenido, como títulos, párrafos, enlaces, imágenes y otros elementos, utilizando etiquetas que los organizan y describen
